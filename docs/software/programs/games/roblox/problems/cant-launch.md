# 로블록스 실행 안 될 때

## 개요
이 문서는 로블록스가 실행되지 않을 때의 문제 해결 방법을 설명한 문서이다.

## 증상
- 게임 클라이언트가 열리지 않음
- 실행 후 바로 종료됨
- 로그인 화면까지 도달하지 못함

## 해결 방법
1. **인터넷 연결 확인**: 안정적인 네트워크 환경인지 확인한다.
2. **클라이언트 재설치**: Roblox를 완전히 제거 후 재설치한다.
3. **방화벽 및 보안 프로그램 확인**: Roblox가 차단되지 않았는지 확인한다.
4. **운영체제 업데이트**: Windows/Mac OS가 최신 상태인지 확인한다.
5. **관리자 권한 실행**: 클라이언트를 관리자 권한으로 실행한다.
