# 로블록스는 어떻게 작동하는가

## 개요
이 문서는 로블록스의 기본 작동 원리를 설명한 문서이다.

## 내용
로블록스는 클라이언트-서버 구조로 운영되며, 사용자는 게임 클라이언트를 통해 다양한 게임에 접속할 수 있다.  
게임은 Roblox Studio로 제작되며, 서버는 게임 로직과 데이터를 처리하고, 클라이언트는 화면과 입력을 담당한다.

주요 특징:
- 사용자 제작 게임 지원
- 크로스 플랫폼 실행 (PC, 모바일, 콘솔)
- 가상 아이템과 경제 시스템
